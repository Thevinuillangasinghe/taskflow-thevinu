# TaskFlow

TaskFlow is a lightweight team task management web application developed as part of an SDLC internship assignment.

The application allows users to create workspaces, manage tasks, assign teammates, and track progress using a Kanban-style workflow.

---

# Features

- User authentication
- Workspace management
- Task creation and management
- Task assignment
- Kanban board
- Activity logging
